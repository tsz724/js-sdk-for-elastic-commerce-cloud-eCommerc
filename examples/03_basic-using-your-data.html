03. Using Your Data

Now that we've got the product out, we need to do something with it!

With this example we'll create a really basic product listing.

Code

// Authenticate var _moltin = new Moltin({publicId: '<your public key>'}); _moltin.Authenticate(); // Listen for authentication window.addEventListener('MoltinReady', function() { // Get your first product var _product = _moltin.Product().Get(<your product id>); // Update our listing var item = $('#example .item'); item.find('.list-group-image').attr('src', _product.images[45].url.http).attr('alt', _product.title); item.find('.list-group-item-heading').html(_product.title); item.find('.list-group-item-text').html(_product.description); item.find('.lead').html(_product.pricing.formatted.without_tax); item.find('.btn-success').on('click', function(e) { alert(_product.title+' added to cart!'); }); });
